# Trademark Policy

## Policy Statement
Vespid source code may be used under applicable open source licenses, but Vespid trademarks, logos, and brand identifiers are not granted by those code licenses.

## Allowed Uses
- Accurate, non-confusing descriptive references (for example: "compatible with Vespid Community").
- Unmodified attribution required by license/NOTICE obligations.

## Prohibited Uses
- Using "Vespid" name/logo in product branding that implies official affiliation.
- Registering confusingly similar names, logos, or domains.
- Using official brand assets in paid offerings without written permission.

## Fork Guidance
- Forks must remove official logos and protected brand marks.
- Fork distributions must clearly state they are independent and unofficial.

## Permission Requests
Commercial branding permissions require written approval from Vespid legal/contact owner.
