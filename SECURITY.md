# Security Policy

## Supported Versions
Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability
Please report vulnerabilities privately.

- Email: security@vespid.ai
- Include: impact, affected components, reproduction steps, and suggested mitigations if available.

Do not open public issues for suspected vulnerabilities until triage is complete.

## Response Targets
- Initial acknowledgement: within 3 business days
- Triage decision: within 7 business days
- Fix timeline: depends on severity and exploitability

## Disclosure
After a fix is available, maintainers coordinate disclosure details and upgrade guidance.
